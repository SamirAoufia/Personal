Lego's Colour

Previous Next

For our multidisciplinary project 2, I was teamed up with Theo, we had to develop a system to read the color of lego blocks and send the date to a database, which we then used to display on our website.

This project involved close collaboration with Theo within our group, where we successfully implemented a solution comprising a color sensor, a microcontroller, a raspberry and a database.

The fundamental process involved the color sensor identifying the color of a lego block, the microcontroller then processing this information and transmitting it in real time to a database. The stored data could then be accessed via our website, providing users with instant updates on color readings

During this experience, I improved my skills in the internet of things (iot) and real time data processing. Working with Theo strengthened my ability to work effectively in a team environment, solve complex problems and oversee a project from conception to completion. Practical challenges included the seamless integration of sensors and the creation of a user- friendly interface.

Overall, this project strengthened my adaptability and problem-solving skills, but also showed me that I have difficulty to work in an open space with several teams because it was noisy.
