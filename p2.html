Lego's Colour

Previous Next

For my second multidisciplinary project, I was teamed up with Theo, we had to develop a system to read the color of lego blocks and send the date to a database, which we then used to display on our website.

This close collaboration with Theo within our group resulted in a successful solution, integrating a color sensor, a microcontroller, a Raspberry Pi and a database.

The key process involved the color sensor identifying the hue of a Lego brick, the microcontroller then processing this information and transmitting it in real time to a database. The stored data was then accessible via our website, providing users with instant updates on color readings.

During this experience, I honed my skills in the Internet of Things (IoT) and real-time data processing. Working with Theo strengthened my ability to collaborate effectively in a team environment, solve complex problems and oversee a project from conception to completion. Practical challenges included the seamless integration of sensors and the creation of a user-friendly interface.

Overall, this project strengthened my adaptability and problem-solving skills.

For this project we used HTML, Bootstrap, JavaScript, NodeJS, Twig, Node-Red, MySQL, Docker and SQL
